# Agent Capabilities & Limitations

## Orchestrator (MCP Server)
- **Capabilities**: Can sequence agent operations and handle errors.
- **Limitations**: Does not generate content itself, strictly passes data.

## Trip Generator
- **Capabilities**: Can map predefined interests to generic activities.
- **Limitations**: Rule-based only. Does not know real-world geography or actual operating hours of places.

## Narrator
- **Capabilities**: Formats text nicely with Markdown and emojis.
- **Limitations**: Static templates.

## Security Agent
- **Capabilities**: Basic type checking, bounds checking, and simple string replacement.
- **Limitations**: Cannot perform complex adversarial analysis.
